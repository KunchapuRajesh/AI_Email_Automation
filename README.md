# AI Email Automation

AI-powered email assistant built using React JS, Spring Boot, Gmail API and Grok AI.

## Features

* Gmail OAuth Login
* Inbox View
* Email Summarization
* AI Reply Generation
* Email Classification
* Priority Detection
* Daily Digest

## Tech Stack

Frontend:

* React JS
* Material UI

Backend:

* Spring Boot
* Gmail API
* Grok AI

## Run Frontend

npm install

npm start

## Run Backend

mvn spring-boot:run
