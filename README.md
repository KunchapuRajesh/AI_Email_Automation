# AI Email Automation

AI-powered Email Automation System using React JS, Spring Boot, Gmail API and Groq AI.

## Features

- Gmail OAuth Login
- Inbox Management
- AI Email Summary
- AI Reply Generation
- Email Classification
- Priority Detection
- Daily Digest

## Tech Stack

### Frontend
- React JS
- Vite
- Axios

### Backend
- Spring Boot
- Gmail API
- Groq AI

## Run Backend

```bash
mvn spring-boot:run
